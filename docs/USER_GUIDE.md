## 2. God Mode: The "Killer Feature"

This is why we built the app. **God Mode** aggregates disjointed data into a single, cohesive narrative.

![God Mode View](./images/god_mode_view_1765208572688.png)

### Why is this better?
Normally, to get this data, you would need to open 5 different charts. Here, it is in one row.
*   **Today Str (Strength)**: Tells you intraday momentum.
*   **P1/P2/P3**: Shows history. Consistently Green = **Strong Uptrend**.
*   **Visual Cues**: Green backgrounds mean "Buy", Red means "Sell". No guessing required.
